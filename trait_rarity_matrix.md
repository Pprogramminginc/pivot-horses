# Trait Rarity Matrix

This file reflects the current trait set the app should use for generated foals.

Important note:
- This is the breeding logic source of truth.
- Some sample horses in the app may still carry older trait assignments until we fully refresh that data.
- `coat_color`, `coat_pattern`, and mutant-as-a-trait are intentionally removed for now and will return later with visual updates.
